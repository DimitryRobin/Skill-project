// Form A propos : 73-93
            
À propos
Retour
Skill project est une application créée, entièrement par moi-même, afin de répertorier toutes les compétences
que j'ai acquises jusqu'ici. L'idée de créer ce petit logiciel m'est venu lorsque j'ai appris qu'il fallait garder
toutes traces de réalisation pratique, lors de mon enseignement en BTS, pour passer l'épreuve E6.
Dès lors, je me suis dit : pourquoi pas créer un portefeuille de compétences sous forme logiciel ?
Après approbration de mes enseignants quant à la possibilité d'utiliser cette méthode pour l'examen, je me
suis mis à la tâche. Puis, je me suis rendu compte, au fur et à mesure que je développais, que Skill project
pourrait servir à autrui.
Je me suis alors posé la question : pourquoi pas faire évoluer l'application afin d'aider mon prochain ?
De part ces raisonnements, Skill project a vu le jour puis évolue petit à petit dans le but d'offrir les réponses aux
questions de tous développeurs, mais aussi dans le but d'offrir du contenu différent de celui trouvé sur internet.
De part sa simplicité d'utilisation et son interface conviviale, Skill project permet :
- d'apprendre, en copiant d'un simple clic, le code d'une compétence
- de faire des suggestions (amélioration du logiciel, ajout de nouvelles compétences etc..)
- de partager mes compétences et mon savoir-faire avec vous
Enfin, Skill project ne nécessite aucune connexion à une base de donnée, premièrement car il n'existe aucun
hébergeur, à ma connaisance, d'application C# (mis à part db4free.net, environnnement de test) et dans un
second temps parce que la connexion à une base de donnée, via des requêtes, pourrait ralentir le logiciel.
Afin de gérer les préférences de chaque utilisateurs, il est possible qu'un fichier .txt soit créée sur votre
ordinateur dans le seul et unique objectif de sauvegarder vos paramètres.
